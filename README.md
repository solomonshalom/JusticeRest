# JusticeRest

The official, ultra-minimal platform of JusticeRest! A platform to Speak Up, Seek Change, and Inspire Justice!

## Credits

Based on [Bublr](https://bublr.life)!

## Installing

```bash
git clone https://github.com/solomonshalom/JusticeRest.git
cd JusticeRest
npm install --force
```

## Building

```bash
npm run build
```

## Developing

```bash
npm run start
```
